Microsoft purchases Github

What is HTTPS?

The Microsoft CEO talked about the details of acquisitions. There are three things about GitHub's future. First, developer first: Focus on developers, developers on GitHub. Second, take advantage of Microsoft to help GitHub achieve revenue growth, improve functionality, and make better support for enterprise users. Third, with this acquisition, Microsoft will achieve a value growth for itself. The main purpose of Microsoft is to build trust, which is the trust of non-Microsoft developers and the Microsoft enterprise itself. By keeping GitHub high quality and high independence, more people will see the side of Microsoft's current development. For investing in the future, it is the developers themselves. In the future, every company will become a software company, there will be more and more non-tech developers, relying on GitHub, Microsoft can lay out the future, and developers tools will become more important than now. With trust, it is good for Microsoft, for the industry, and for everyone.
