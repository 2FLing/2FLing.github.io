Differnce between Java and Javascript

What is Javascript?

JavaScript is a script or programming language that allows you to implement complex functions on a Web page -- not just static information every time you browse a Web page -- to display up-to-date content, or interactive maps, or 2D/3D graphical animations, or to automatically play video, and so on. You can be sure, JavaScript is involved. This is the three level of cake standard for Web technology, two of which are HTML and CSS.

What is Java?

Java is the programming language and computing platform first published by Sun Microsystems in 1995. There are many applications and Web sites that work only after Java is installed, and such applications and Web sites are growing. Java is fast, safe and reliable. From laptops to data centers, from game consoles to science supercomputers, from mobile phones to the Internet, Java is everywhere!
