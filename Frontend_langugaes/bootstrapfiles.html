Files in Bootstrap

Inventory of the files and folder

startbootstrap-resume-gh-pages/css

- That is for storing the css file of the template.

startbootstrap-resume-gh-pages/img

- That is for storing the image of the template.

startbootstrap-resume-gh-pages/js

- That is for storing the files of javascript of the template.
- There are two different file extension name there:
- .js:JS is the abbreviation of javascript. - function: JS is a web page foreground script, almost all you see web page effect is the credit of JS, without JS web page will become rigid. - .min.js:.min.js is the compressed version of the JS file.

startbootstrap-resume-gh-pages/scss

- That is for storing the files of scss of the template. - .scss: CSS is not a programming language. So, people start adding programming elements to CSS, which is called a "CSS preprocessor". Its basic idea is to use a special programming language, web page style design, and then compile into a normal CSS file. Sass is one of the preprocessor, and SCSS is the abbreviation of SASS CSS.

startbootstrap-resume-gh-pages/vendor

- That is for storing the third party library

startbootstrap-resume-gh-pages/vendor/bootstrap

- That is for storing the bootstrap stuff from third party library

startbootstrap-resume-gh-pages/vendor/fontawesome-free/less

- 'less' file: LESCSS(Leaner Style Sheets) belongs to CSS preprocessing language. It uses CSS-like syntax, which gives CSS dynamic language features, such as variables, inheritance, operations, functions, and so on. It is more convenient to write and maintain CSS.

startbootstrap-resume-gh-pages/vendor/fontawesome-free/sprites

- That is for storing the files of .svg - '.svg': SVG(Scalable Vector Graphics) is an open standard vector graphics language, which is a scalable vector graphics format.Svg format images can be enlarged arbitrarily, and the edges are very clear.The generated files are small enough to facilitate transmission.

startbootstrap-resume-gh-pages/vendor/fontawesome-free/webfonts

- That is for storing the fonts the template would use.

startbootstrap-resume-gh-pages/.json

- .json:JSON (JavaScript Object Notation) is a lightweight data interchange language. It is based on easy-to-read text and is used to transfer data objects composed of attribute values or sequential values. Although JSON is a subset of JavaScript, but JSON is a language-independent text format also, and has the habits similar to those of the C language family.
