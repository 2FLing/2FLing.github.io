About HTML

HTML is the standard markup language for creating Web pages. HTML stands for Hyper Text Markup Language. HTML describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. HTML elements are represented by tags.

How to Learn HTML

Suggest not using Dreamweaver at first, or you may fall into the trap of many "Must Learn XXX", "Master XXX X X on X Day" and videos of uncles who speak less standard Mandarin (it seems that many people are black here.) Returning to the main topic, I suggest buying a textbook for novice HTML learners, learning the meaning and usage of each tag in HTML, recommending "The Road to HTML: A Guide to Best Practices for XHTML and CSS". This book is a collection of the best content in the famous foreign website HTML DOG. It can be completely zero-based, Zero-start systematic learning of HTML and CSS. The most important thing is that it is very, very thin, and it does not seem to be mentally laborious. Some teaching websites are also helpful, such as the w3school online tutorial, which can be used as a reference manual. If English isn't too hard, look at the HTML foundation tutorial in Learn to code, the interactive interface, and the badge reward after completing the task correctly. It's very motivating to do so after completing the above steps. It's also interesting to think more about the semantic structure, layout, color matching, how those dazzling dynamic effects are made, what new libraries are used, what new frameworks are used, and how to validate your ideas with the browser's View Source Code. Teacher, take a small task, bit by bit to complete, in the process to learn to use the good thing http://www.google.com to overcome their inertia, cultivate their interest in HTML and good Luck!
